WP REST API - Author Info Plugin
=================

This plugin will get Author's Display Name for each Post in WP REST API.

It takes this:


	author: 1,


And turns it into this:


	author_info: {
		id: 1,
		display_name: "AUTHOR_DISPLAY_NAME"
	},
